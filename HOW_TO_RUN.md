# 🚀 How to Run — IT Yearbook 2026

## Step 1 — Copy the folder to your Desktop

Move the `it-yearbook-2026` folder to your Desktop (or anywhere you like).

---

## Step 2 — Open in VS Code

Right-click the folder → **Open with Code**
(or open VS Code → File → Open Folder → select `it-yearbook-2026`)

---

## Step 3 — Open the Terminal in VS Code

Press:  `` Ctrl + ` ``  (backtick)

Or go to: **Terminal → New Terminal**

---

## Step 4 — Install dependencies

```bash
npm install
```

This downloads React, Framer Motion, TailwindCSS, and all other packages.
It takes about 30–60 seconds the first time.

---

## Step 5 — Start the dev server

```bash
npm run dev
```

You'll see output like:
```
  VITE v5.x.x  ready in 300ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: http://192.168.x.x:5173/
```

---

## Step 6 — Open in browser

Click the link or go to: **http://localhost:5173**

The yearbook will open with the loading screen! 🎓

---

## Adding Photos

See `IMAGE_GUIDE.md` for exactly where to put student portraits,

## Building for Production (optional)

```bash
npm run build
```

Output goes to the `dist/` folder — upload that anywhere (Netlify, Vercel, GitHub Pages).
